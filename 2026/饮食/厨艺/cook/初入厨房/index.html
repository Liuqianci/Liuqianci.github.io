一、初入厨房

发表于 2026-06-09 更新于 2026-07-18 分类于 饮食 ， 厨艺 阅读次数：
本文字数： 2.5k

基本厨具

  • 锅具
    • 炒锅：万能的，煎炒烹炸样样精通；
    • 汤锅：用于煮面条或者烧水；
    • 平底不粘锅：用于煎饼、煎肉；
    • 小奶锅：做快手菜
  • 菜刀
    • 切刀：刃薄，锋利，适合切菜；
    • 砍刀：刃厚，稍微钝，适合砍骨头；

家中必备调理

  • 生抽
  • 陈醋
  • 油
  • 大蒜
  • 生姜
  • 淀粉
  • 鸡精
  • 味精
  • 盐

以上9个调料可以应付80%的家常菜。可以适当添加以下几种扩充：

  • 料酒
  • 豆瓣酱
  • 干辣椒
  • 白糖
  • 胡椒粉

认识盐

  盐是百味的基础，做菜时别的调料可以不放，但绝对不能少了盐。放盐的原则：宁少勿多！！

煎鸡蛋

  1. 碗中打入两个鸡蛋，加入适量食盐，一个鸡蛋就是一个小指甲盖那么多的盐；
  2. 用筷子把鸡蛋打散。左手拿碗可以稍微向右手倾斜，不容易撒出去。打10多秒即可，目的是让蛋清和蛋黄混在一起；
  3. 开火，把锅中水分烧干；
  4. 倒油，用油把锅底润一下；
  5. 油冒烟后，倒入鸡蛋；
  6. 鸡蛋蛋清凝固后记得用锅铲捅破，让里面蛋黄流出来，否则里面煎不到；
  7. 蛋液定型后翻面，一面不要煎太久，多操作几次，两面金黄即可捞出；

注意，油少或者油温过高，容易煎糊；油温太低，煎出的蛋不够松软。

荷包蛋

  1. 小火，倒入少许的油；
  2. 油在微微冒烟时，打一个鸡蛋下去，让鸡蛋在油里慢慢的煎，等定型（边缘变的焦黄）后翻面；
  3. 撒上少许盐，倒在手掌上分开撒，再翻面；
  4. 多翻面，让每一面都接触到盐，然后盛出；

  喜欢溏心蛋的可以少煎一会或者只煎一面，不过从营养学的角度来说不建议吃溏心蛋，溏心蛋没有熟，不利于人体吸收。全熟蛋杀菌效果更彻底，更利于健康。

味精与鸡精

  鸡精和味精主要用于调味，放很少一点就可以成倍的提升菜品的口味。不过很多人认为鸡精和味精有害，尤其是味精在加热到120度后会生成焦谷氨酸钠，可能致癌。但有论文指出适量吃鸡精和味精都是安全的。只需要注意味精温度过高会失去鲜味，所以只在出锅前加味精。

炒空心菜——味精和鸡精的运用

  1. 择菜——把食材处理的长短大小统一，方便食用，方便烹饪；把叶子都摘下来，如果空心菜不是很老，那么根部也可以留着，只要用指甲掐一下根部，能掐动就代表还不老；
  2. 洗菜，把泥沙充分洗掉，淘洗就行，不要用手搓；洗完用滤网捞出来；
  3. 开火，直接开最大的火，把锅里水烧干；
  4. 倒油，对于铁锅来说宁多勿少防止炒糊；
  5. 油开始冒烟后倒入蔬菜，动作一定要快，用菜把油盖住防止砰溅；绿叶菜很容易熟，所以需要经常的翻炒，每隔几秒钟就翻炒一下；这个就看火大小，火大勤翻炒，火小慢翻炒；
  6. 叶子快蔫掉时代表快熟了，此时放调料，放入盐、鸡精、味精，撒开一点。盐要多放一点（因为蔬菜的水多），鸡精和味精用量是盐的1/4~1/5即可，鸡精可以更少；放入调料后疯狂翻炒，让调料均匀；

鸡精和味精一起使用有1+1>2的效果。

用旺火炒菜的原因：用旺火炒菜更好吃，且蔬菜中有很多的维生素被加热后容易流失，所以用旺火炒菜可以尽快的出锅。

炒菜的第一个公式

油+菜+盐+鸡精\味精 -> 出锅。先把油烧热，放入菜，菜快熟的时候放入盐、鸡精、味精，把调理和匀就可以出锅了。至少80%带叶子的蔬菜都可以这么做。

翻炒时注意，如果菜的茎叶比较厚实，类似白菜，就可以适当减少翻炒的频率。

放调料的时候可以先放到锅铲上，这样可以控制用量，也方便和均匀。

根据经验来看，有些蔬菜含水量很大，炒菜后的汤水会很多，这时候要么选择把汤汁倒掉，要么就多放盐。

干辣椒、大蒜、花椒

干辣椒、大蒜、花椒也属于调料的范畴，不过他们是土里长出来的天然调料，一样可以增加食品的风味。这三个是川菜中的三剑客，经常形影不离。

新手炒菜容易把这些配料炒糊，因为油温太高了。在熟能生巧之前，可以在放入油后立马下锅配料。

炒菜苔

  菜苔择菜的时候需要一点小技巧，用一把刀从根部切十字，第一刀切到第二片叶子的位置，第二刀切到第三片叶子的位置。然后从第三片叶子稍微往上一点的地方切掉，这样就是一个十字花刀的形状，方便炒熟也方便入味。剩下的部分很细，切一刀就可以了。菜苔上有花的话把花去掉，口感不好。如果买来的菜苔根部很老的话，还需要先去根部的皮。

  一盘菜用三颗大蒜，三个干辣椒，几粒干花椒：

  • 用到把干辣椒切成一厘米长的段；
  • 大蒜剥皮，剥皮最快的方法就是用菜刀拍。然后把大蒜切成片。

炒菜步骤：

  1. 烧半锅水，水开后把菜苔焯水（菜苔的茎太厚了，不好炒，需要提前焯水），煮1分钟到1分半即可，不要太久，否则就不脆了；
  2. 把菜苔倒入滤网中，用冷水冲，可以保持爽脆口感；
  3. 锅烧干后倒油，然后马上放入准备好的干花椒、干辣椒；看到干辣椒微微发黑的时候放大蒜，然后立即放入准备好的菜；
  4. 提前焯水后菜基本已经熟了，所以翻炒片刻就可以放盐、味精、鸡精，和匀后出锅；

 

手撕莲白

洋葱、大白菜、卷心菜等都需要去掉外边1到2层，之地很老，卖相差，口感也不好。

然后把莲白一层一层的拨下来，每一片叶子在随意撕成片。手撕的比刀切的好吃，因为手撕的断面不整齐，更容易入味。只要叶子部分，菜帮子可以丢掉。撕完之后清洗滤水。

依旧准备大蒜、干辣椒、干花椒。莲白菜不需要焯水。

 

素炒四季豆

四季豆的挑选：粗壮的四季豆口感稍微老，但是四季豆味道足，缺点是筋多；细长的四季豆口感更细嫩。挑选时看表面有没有黑点或者坑坑洼洼的，这可能被虫子咬过。

四季豆比较耐放，在冰箱中放7~10天没问题。

四季豆择菜时要把两侧的筋去掉，从头部的尖开始，逆着尖使劲往后一拉就行，然后把尾巴掐下一点，逆着把另一侧筋去掉。两侧筋都去掉后，把四季豆恰成5厘米左右的段，这时中间一些没有去掉的筋就露出来了，可以去掉。

四季豆没有泥沙，清洗时使劲拿手和一下就行。

素炒四季豆的配料，可以准备2-3瓣大蒜切片。还有一个秘密武器：留兰香（食用薄荷），取顶端的部分，掐5-6根左右，用刀稍微切碎即可。

生的四季豆吃了会引起食物中毒，所以四季豆的做法要分两步：先制熟，然后回锅调味。

在饭店里制熟四季豆，用的是大量热油，又快又好吃，但是家里这样做太浪费了。家里比较经济实惠的方法就是用水煮，把四季豆放入锅中，加水，水要稍微没过四季豆。盖上盖子大火烧开，然后转中火再煮5~10分钟。出锅后用凉水冲一下口感更佳。

剩下的步骤很简单：大火烧干锅，放油、放入大蒜，放入四季豆翻炒。不过四季豆很难入味，所以需要更早的时候放入盐，然后再翻炒两三分钟。最后调为中火再煎两三分钟。出锅前放入鸡精、味精、食用薄荷，翻炒几秒把薄荷炒到打蔫即可出锅。

豇豆、扁豆都可以用这个方法来做。
